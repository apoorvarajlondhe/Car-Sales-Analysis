# Car-Sales-Analysis

Designed and developed an interactive Car Sales Analysis dashboard using Power BI to track sales performance, revenue trends, and key KPIs.

Implemented data modeling, created relationships, and performed data cleaning using Power Query for accurate reporting.

Built visually compelling charts such as sales trend line charts, region-wise performance maps, model-wise comparison charts, and KPI cards.

Used DAX measures (YTD Sales, Profit %, Average Selling Price, Growth Rate) to derive actionable insights.

Added slicers and filters (Year, Car Model, Region, Fuel Type) to enhance interactivity and user-driven exploration.

Applied drill-through and drill-down features for deeper analysis of car brands and dealership performance.

Enabled automated insights using Smart Narrative and Key Influencers to highlight top-performing models and factors affecting sales.

Ensured optimal user experience by designing a clean, responsive layout suitable for Power BI service and mobile view.

Published the report to Power BI Service, set up a workspace, and configured scheduled refresh for automated data updates.

Delivered insights that helped identify high-demand models, seasonal trends, and underperforming regions.
